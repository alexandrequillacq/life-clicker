# Études de cas — jeux de référence

Ce que chaque jeu nous apprend, et **ce qu'on lui vole**. Source : deep research du 2026-06-21 (16 jeux).

| Jeu | Ce qui le rend fort | Ce qu'on vole |
|---|---|---|
| **Universal Paperclips** | Démarrage 1 bouton ; 3 actes qui changent de verbe ; tu réalises que *tu es* l'IA ; fin avec choix ; ~5h bornées. | Le modèle quasi-complet : révélation progressive + verbes par phase + twist diégétique + fin avec choix. |
| **A Dark Room** | Écran noir, 1 bouton ; change de genre 4× en silence ; twist « tu n'es pas humain ». | La révélation par seuils + le twist non-annoncé. |
| **Candy Box** | Pionnier du « rien puis ça grandit » ; UI qui apparaît par paliers ; brise le 4e mur. | La mécanique de révélation d'onglets/features par seuils. |
| **Spaceplan** | Récit en blips d'humour entre les actions idle ; boucle temporelle ; *a une fin*. | Le récit qui n'interrompt jamais le chiffre qui monte. |
| **Cookie Clicker** | Le cœur du genre : coût ×1,15, upgrades qui doublent, Golden Cookies, succès→lait→production, prestige cube-root. | Les maths de coût/plaisir + bonus aléatoires + prestige. |
| **AdVenture Capitalist** | Managers (automatisation, même hors-ligne) ; paliers 25/50/100… ; anges = √. | Automatisation des clics + paliers de doublement. ⚠️ *aussi le contre-exemple pay-to-win.* |
| **Antimatter Dimensions** | Prestige empilé (Infinity→Eternity→Reality), chaque couche = nouvelle mécanique ; ~825h. | Le modèle du prestige multi-couches si on veut de la profondeur. |
| **Kittens Game** | Phases nombreuses (village→science→religion→espace→temps) ; horloges longues. | L'idée de phases thématiques enchaînées. |
| **Egg Inc** | Hors-ligne plafonné ~2h ; Soul Eggs ; prestige fréquent récompensé. | Le plafond hors-ligne + courbe de prestige qui récompense le retour. |
| **NGU Idle** | Chaque boss débloque un sous-système entier ; multi-milliers d'heures. | Débloquer des systèmes entiers comme récompense de palier. |
| **Leaf Blower Revolution** | 94 % positif ; premium 100 % gagnable, rien ne gate. | ✅ Le modèle de monétisation sain (ou pas de monétisation). |
| **Idle Dungeons** | Échec (31 %) : prestige « ne récompense rien », resets gonflés pour vendre des tokens. | ❌ Le contre-exemple absolu : prestige punitif + pay-to-win. |
| **Cell to Singularity** | Thème évolution (cellule→IA→Singularité) ; mais hors-ligne radin → révolte. | Le thème d'évolution par ères ; ❌ le hors-ligne cassé à éviter. |
| **Clicker Heroes** | « On revient à une liste de récompenses, pas une to-do » (hors-ligne). | La philosophie du progrès hors-ligne. |

## Théorie de design citée
- **« The Math of Idle Games »** (Pecorella / Kongregate) : coût ×1,15 vs prod ×1,1, seuil de perception ×1,2, le prestige comme « soft cap ».
- **Frank Lantz** (Paperclips) : le clicker te fait vivre l'obsession d'un but arbitraire.
- **Benchmarks GameAnalytics** : session ~8 min, ~5/jour, cycle Hook/Habit/Hobby.
